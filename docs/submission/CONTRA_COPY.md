## Post

```text
I built Lesson Loom — Plan to Playable for the Google Stitch Challenge.

The prototype turns a teacher-approved lesson plan into an interactive classroom interface. The demo starts with a plain Grade 5 equivalent fractions lesson and transforms it into Fraction Garden: a student activity, teacher console, differentiation supports, assessment checkpoints, printable fallback, and Stitch-ready export pack.

The goal was not to create another AI lesson-plan generator. It was to explore what happens when Stitch-style UI generation is used for classroom materials: static plans become rich, reviewable learning interfaces, while the teacher stays in control.
```

## Process note

```text
Built as a high-fidelity prototype using Stitch for UI direction and Cursor/Composer for implementation. The design system uses warm paper textures, deep navy typography, garden-inspired fraction visuals, and a woven signal motif to show the lesson transformation.

The prototype keeps trust visible: teacher review, draft alignment, no student data, printable fallback, UDL-informed supports, and no automated grading.
```
